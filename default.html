Large Language Model

LLM = Large Language Model

by Charles Iliya Krempeaux

A Large Language Model (LLM) is a type of machine learning and artificial intelligence (AI) model that is usually based on the deep learning transformer architecture, that is trained on large amounts of text to predict the next token in a sequence.
