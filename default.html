Large Language Model

LLM = Large Language Model

by Charles Iliya Krempeaux

A Large Language Model (LLM) is a type of machine learning and artificial intelligence (AI) model that is usually based on the deep learning transformer architecture, that is trained on large amounts of text to predict the next token in a sequence.

Some tasks an LLM can be used to do are: text generation, question answering, summarization, translation, source-code generation, and reasoning-like behaviors without being explicitly programmed for each task (beyond the initial training).

At least at the time of writing, most people have probably experienced LLMs via AI) chatbots and agentic computing.

LLM Architectures

There isn't just one type of LLM. There are a number of different LLM architectures.

Some of the different LLM architectures are:

Autoregressive (Decoder-only) Models
These types of LLMs are good at generating text by predicting the next word in a sequence, one by one. These types of LLMs are useful for generating text. Examples of this type of LLM are GTP and LLaMA.
Autoencoding (Encoder-only) Models
These types of LLMs are designed to "understand" and analyze text (rather than generate text from scratch). These types of LLMs are useful for classification or sentiment analysis. An example of this type of LLM is: BERT.
Sequence-to-Sequence (Encoder-Decoder) Models
These types of LLMs combines both encoding and decoding. An example of this type of LLM is: T5.
